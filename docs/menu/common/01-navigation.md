# 공통 — 네비게이션

## 요건

- 전역 메뉴: **채팅**(기본), **소스**, 도구, 설정.
- 기본 라우트는 채팅.
- 반응형에서 접이식 메뉴.

---

## 디자인 톤 (노트북LM 참고)

- **레이아웃**: 3단(세션·대화·스킬/소스). 여백·패딩을 넉넉히.
- **카드**: 둥근 모서리(rounded-xl), 얕은 그림자 또는 테두리로 구역 구분.
- **타이포**: 제목·본문 계층 명확. muted 색으로 보조 텍스트.
- **버튼**: 주 액션은 채도 있는 색, 보조는 outline 또는 ghost.
- **빈 상태**: 안내 문구와 아이콘으로 "무엇을 할 수 있는지" 설명.
