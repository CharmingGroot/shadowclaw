# 소스(Sources) — 개요

노트북LM 스타일의 **소스 연결** 기능. 사용자가 참조할 문서·URL·텍스트를 등록하고, 채팅 시 에이전트가 해당 소스를 컨텍스트로 활용해 답변한다.

---

## 역할

- **소스 추가**: 파일 업로드, 웹 URL, 텍스트 붙여넣기로 "소스" 등록.
- **소스 목록**: 등록된 소스 목록 표시(이름, 유형, 삭제).
- **채팅 연동**: 세션 또는 현재 대화에 "사용할 소스"를 지정. 에이전트는 지정된 소스 내용을 참고해 답변.

---

## 소스 유형(목표)

| 유형 | 설명 | 비고 |
|------|------|------|
| **파일** | PDF, TXT, 이미지, 오디오 등 업로드. 텍스트 추출 또는 메타 저장. | 구현 시 파일 크기·형식 제한 |
| **URL** | 웹사이트 URL 입력. 스크래핑/요약 결과를 소스로 저장. | 도메인·안전 정책 적용 |
| **텍스트** | 사용자가 붙여넣은 텍스트를 소스로 등록. | 즉시 사용 가능 |

확장: 웹 검색 결과, Google Drive 등 외부 스토리지는 별도 기획.

---

## UI 요건 (노트북LM 참고)

- **위치**: 채팅 화면 좌측 패널(세션 목록 아래/위) 또는 별도 "소스" 탭. 또는 채팅과 동일한 3단 레이아웃에서 "출처" 전용 패널.
- **소스 추가**: "+ 소스 추가" 버튼 → 모달 또는 인라인 폼. 유형 선택(파일/URL/텍스트) 후 입력·업로드.
- **소스 목록**: 카드 또는 리스트. 제목(또는 파일명/URL 요약), 유형 뱃지, 삭제 버튼. 선택 시 "이 대화에 연결" 체크 또는 드래그.
- **빈 상태**: "저장된 소스가 여기에 표시됩니다", "PDF, 웹사이트, 텍스트를 추가하거나 붙여넣으세요" 등 안내 문구.
- **디자인**: 노트북LM처럼 깔끔한 카드·둥근 모서리·명확한 계층. 검색 바(선택)로 소스 필터.

---

## 연동

- **세션별 소스**: 세션 생성 시 또는 채팅 화면에서 "이 대화에 사용할 소스"를 선택. ReAct/LLM 호출 시 해당 소스 내용(또는 요약/임베딩 검색 결과)을 프롬프트에 포함.
- **글로벌 소스**: (선택) 앱 전역 "기본 소스" 풀을 두고, 세션에서 그중 일부를 선택해 연결.
