# 도구 — MCP

## 기능

- MCP 서버 등록(URL 또는 stdio 설정).
- 연결/해제, 상태 표시.
- 등록된 서버의 도구(Tools) 목록 조회·노출.
- 채팅 시 해당 도구 호출 가능하도록 연동.

## UI 요건

- **서버 목록**: 등록된 MCP 서버 목록 표시. 행별 연결 상태, 도구 펼치기, 삭제 버튼.
- **서버 추가**: 목록 상단(또는 섹션 헤더)에 **「MCP 서버 등록」** 버튼만 노출. 클릭 시 **모달(팝업)** 을 띄우고, 모달 안에서만 등록 폼 입력.
  - 모달 내용: 이름(선택), URL 또는 stdio 경로, 전송 방식(http/stdio) 등. [취소] / [등록] 버튼.
  - 등록 완료 시 모달 닫기, 목록 새로고침. 인라인 입력 없음.
- **도구 목록**: 서버 행 펼치기 시 해당 서버의 도구 목록(이름, 설명).
