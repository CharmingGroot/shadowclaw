# ShadowClaw — 프로젝트 역할 요약

지금까지 작성된 기획을 바탕으로, **이 프로젝트가 수행할 역할**을 한 문서로 정리한 것이다.  
상세 요건은 [PRD.md](PRD.md), [menu/README.md](../menu/README.md), [builtin-tools-and-skills.md](../menu/tools/builtin-tools-and-skills.md) 참고.

---

## 1. 이 프로젝트가 하는 일 (한 줄)

**채팅이 메인인 개인 AI 어시스턴트**로, 사용자 말을 받아 **ReAct·도구(Skill/MCP) 호출**로 일을 처리하고, **MCP·Skill을 자유롭게 등록·연결·관리**할 수 있게 한다.

---

## 2. 핵심 역할

| 역할 | 설명 |
|------|------|
| **채팅 중심 인터페이스** | 앱의 메인 화면은 채팅. 세션 목록(좌) + 대화(중앙) + 스킬/도구 패널(우). 사용자는 여기서 지시하고 응답을 받는다. |
| **OpenClaw 방식 일처리** | 사용자 메시지 → **ReAct 루프**(생각 → 도구 선택 → 실행 → 관찰 → 반복 또는 답변). 내부 Skill 또는 **MCP 클라이언트로 외부 MCP 도구** 호출. |
| **MCP·Skill 연동·관리** | **외부 MCP 서버**를 URL/stdio로 등록·연결·해제하고, 그 서버의 도구 목록을 조회·노출. **내장/등록 Skill**은 목록·설명·파라미터·활성 여부 관리. 채팅 시 에이전트가 이 “도구 풀”을 사용. |
| **API Key 안전 관리** | 설정 화면에서 LLM API Key 입력·저장(암호화·마스킹). 미설정 시 채팅 불가 또는 안내. |
| **세션 관리** | 대화 단위로 세션. 생성·전환·삭제·제목 수정. 세션별 히스토리 유지. |

---

## 3. 기술적 역할

| 구분 | 내용 |
|------|------|
| **스택** | **TypeScript** + **Node.js**. 에이전트 API, ReAct, MCP 클라이언트, 세션·도구 관리 모두 Node 서버에서 수행. |
| **내장 도구** | 파일시스템(읽기/쓰기/목록/존재), 리눅스·쉘(명령 실행), 스킬(조회·메타 수정·커스텀 생성). 각각 **Skill로 래핑**해 에이전트가 호출. |
| **외부 MCP** | 사용자가 등록한 **외부 MCP 서버**에 연결해, 해당 서버가 제공하는 **도구**를 채팅 중 에이전트가 호출할 수 있게 함. |

---

## 4. 화면별 역할

| 화면 | 역할 |
|------|------|
| **채팅** | 메인. 세션 목록 + 대화 + (우측) 스킬/도구 패널. 메시지 전송 시 ReAct·도구 호출로 응답. 스킬/도구 강제 지정·툴팁. |
| **도구** | MCP 서버 등록/해제/연결, MCP 도구 목록 조회. 스킬 목록·설정·활성/비활성. 채팅에서 쓰는 “도구 풀”을 여기서 관리. |
| **설정** | API Key 입력·저장·안전 관리, 모델·기타 앱 설정. |
| **세션** | (선택) 세션 전용 목록·검색·삭제·제목 수정. 채팅 좌측에 세션 목록이 있으면 생략 가능. |
| **공통** | 네비게이션(채팅/도구/설정), 반응형, 빈/에러 상태, 디자인 톤. |

---

## 5. 정리 — “무엇을 하는 프로젝트인가”

- **제품**: 채팅이 메인인 **개인 AI 어시스턴트 (ShadowClaw)**.
- **일처리**: 사용자 명령을 **ReAct + 도구(Skill/MCP)** 로 실행. 내장 Skill + **외부 MCP 서버 도구**를 하나의 도구 풀로 다룸.
- **차별점**: MCP·Skill을 **등록/연결/해제/설정**할 수 있고, 채팅에서 그 도구를 골라 쓰거나 강제 지정할 수 있음. API Key는 설정에서 안전하게 관리.
- **기술**: TypeScript·Node.js. 내장 도구(파일·쉘·스킬 관리)는 Skill로 감싸서 제공하고, 외부 MCP는 클라이언트로 연결해 도구를 확장.

즉, **“채팅으로 말 걸면, ReAct 에이전트가 내장 Skill + 사용자가 연결한 MCP 도구를 써서 일을 처리해 주는, 개인용 AI 어시스턴트”**를 만드는 프로젝트다.
